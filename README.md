# Visualisierung & Datenaufbereitung — Kurs-Site

## Schnellstart
```bash
# 1) Node 18+ installieren
# 2) Abhängigkeiten installieren
npm i

# 3) Entwickeln
npm run start

# 4) Bauen & (optional) deployen
npm run build
# npm run deploy   # GitHub Pages (config in docusaurus.config.js anpassen)
```
